CustomRoleplayMessages

CustomRoleplayMessages es un plugin de Minecraft que te permite enviar mensajes en el chat con autores personalizados, por ejemplo, como admin.

  • GitHub
  • SpigotMC
  • Modrinth
  • Discord

Funciones

¿Qué funciones tiene este plugin? Algunas básicas, este plugin tiene funciones específicas.

Autores personalizados

¡El autor que quieras! Admin, dioses, servidor.

Colores distintos

Cada comando tiene un color predeterminado distinto.

Comandos

Aquí encontrarás la lista de comandos con sus respectivos permisos requeridos.

Comando Descripción Permisos
/admin [jugador] <mensaje> Envía mensajes como 'ADMIN'. customroleplaymessages.AdminCommand
/dioses [jugador] <mensaje> Envía mensajes como 'DIOSES'. customroleplaymessages.DiosesCommand
/servidor [jugador] <mensaje> Envía mensajes como 'SERVIDOR'. customroleplaymessages.ServidorCommand
